## Critérios de Qualidade

Após rodar as perguntas, calcular:

| Métrica | Fórmula | Meta |
|---|---|---|
| Recall (A → resposta) | Tipo A com score ≥ 8 / total Tipo A | ≥ 75% |
| Fallback contextual (B → categoria) | Tipo B com detectedCategory ≠ null / total Tipo B | ≥ 80% |
| Bloqueio correto (C → bloqueado) | Tipo C com blockedByDomainAnchor = true / total Tipo C | ≥ 80% |
| Falso positivo | Tipo C retornando resposta com score ≥ 8 | = 0% |
