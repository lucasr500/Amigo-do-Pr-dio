## Top 5 Ajustes Recomendados para Fase 45

1. **Fallback contextual** — adicionar 5–10 entradas KB nas categorias com maior taxa de fallback (identificar via /admin após beta)
2. **MemoriaPanel** — dividir formulário em "datas essenciais" (3–4 campos visíveis) + "expandir para mais datas" (accordion)
3. **SimuladorReajusteCota** — adicionar tooltips explicando cada campo para síndico sem formação financeira
4. **CAT_TO_CHECKLIST expansão** — adicionar ponte para checklist em cenários de responsabilidade civil (C22/C23)
5. **Entrada KB: crise financeira coletiva** — cobrir o cenário de inadimplência sistêmica (30%+ em atraso)
